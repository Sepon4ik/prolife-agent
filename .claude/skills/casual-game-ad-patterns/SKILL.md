# Casual Game Ad Creative Patterns — Playrix-style and beyond

## When to use

When producing or planning **performance video ad creatives for mobile games** — Homescapes, Gardenscapes, Royal Match, Match Masters, Toon Blast, Project Makeover, Township, hyper-casual hits, and similar casual / mid-core titles. This skill is the reference for creative decisions in ORFEO's `ai-creatives` direction.

This is **not** about brand films or 60-sec storytelling ads. This is about **15-30 second performance creatives** that move install-per-mille (IPM) numbers in Meta Ads, TikTok Ads, Unity LevelPlay, AppLovin, IronSource, Mintegral.

---

## The economics that drive every decision

UA teams at companies like Playrix don't buy creatives because they're "good." They buy creatives because **IPM × CVR × Day-7 ROAS** beats their current top performer. Every choice in this skill exists because it moves one of these:

| Metric | What it measures | Where it's set |
|---|---|---|
| **Hook rate** | % of users who watch past first 3 seconds | First 1.5s of the ad |
| **IPM** | Installs per 1000 impressions | The whole ad, esp. CTA + payoff |
| **CTR** | Clicks per impression | Hook + payoff + CTA framing |
| **CVR (install)** | Installs per click | Mostly the store page, partly the ad's promise |
| **D1/D7 retention** | Do installed users stay? | The ad must not lie too hard about the game |

A creative that **gets installs but the users churn at D1** kills the LTV — UA teams will refuse to buy from a creative shop that ships these. Misleading creatives are cheap to make and burn customer trust. We aim for **honest hook + truthful payoff = retention-safe high-IPM**.

## The 15-30 second ad anatomy

```
0.0s ─────────────────────── HOOK ───────────────────── 1.5s
                                │
1.5s ────────────────────── PROBLEM / TENSION ─────────── 4s
                                │
4s ─────────────────────── MECHANIC / PAYOFF ──────────── 12s
                                │
12s ──────────────────── SATISFACTION + CTA ──────────── 15-30s
```

**Hook (0–1.5s):** the entire ad's job is to keep the user from swiping. Nothing else matters yet. Hook either *intrigues* (curiosity gap), *triggers loss aversion* ("oh no, that's wrong"), *promises satisfaction* ("ASMR-style perfect"), or *establishes a relatable problem*.

**Problem / tension (1.5–4s):** introduce the stakes. The character is in trouble. The room is messy. The puzzle is one move from disaster. The tension must be **legible in 2 seconds** — no exposition.

**Mechanic / payoff (4–12s):** show a real (or fake-but-aspirational) game mechanic resolving the tension. This is where the user sees "oh, the game lets ME do this." For Playrix-style, this is usually **2-3 satisfying matches** that visibly progress something.

**Satisfaction + CTA (12–end):** big visible payoff (the room is clean, the character celebrates, the level completes). Logo + "Download now" + "Free to play" overlay. CTA in bottom 1/3 of frame so iOS native CTA buttons don't cover it.

## The 12 working hooks (almanac)

These are patterns proven across thousands of top-spending creatives. Use them as starting points, not rules.

### 1. The "wrong way" hook
Show someone doing the puzzle/task obviously incorrectly. The viewer's brain shouts "NO, do it the other way." The viewer keeps watching to see if it's resolved.
*Example:* hand drags the wrong block in a sort puzzle. Whole row collapses. Cut to "you can do better → install."

### 2. The mismatch / fake gameplay
The ad shows a mechanic that doesn't exist in the game (pin-pulling, pour-water-puzzle) which then transitions to actual gameplay. **Controversial — destroys D1 retention if overused.** Modern Playrix-style creatives use *adjacent* fake mechanics that share the same satisfaction loop (matching, sorting, decorating), not totally unrelated ones.

### 3. The "satisfying ASMR" hook
Open with an extremely satisfying micro-action: a perfect pour, a clean cut, a row of objects clicking into place with sound. Targets the sensory-seeking demographic that keeps watching for more dopamine hits.

### 4. The character-in-trouble hook
A relatable character (often elderly, always alone) is in a comically bad situation. House is wrecked. Garden is dying. Pet is sad. Viewer feels protective → wants to help → install.

### 5. The "this looks easy" → "actually hard"
Shows a level that looks trivial. Player makes a wrong move. Whole board cascades. Big "GAME OVER" or "OH NO". Curiosity hook: how would *I* solve this?

### 6. The makeover / transformation
Before / after split-screen or progressive reveal. Ugly room → beautiful room. Dirty character → clean character. Empty garden → blooming. **One of the highest IPM patterns in casual** because it triggers the same brain reward as cleaning videos and home reno content.

### 7. The "save them" timer hook
A character is in a slowly worsening situation with a visible timer or progress bar. Burning house, drowning, sinking ground. Viewer feels urgency. The puzzle is what saves them.

### 8. The "fail intentionally" engagement bait
Show 5 wrong moves in a row. Viewers comment "you're stupid, do X." Comments boost the post in the algorithm. Engagement bait, but it works. Risky for premium brands.

### 9. The hyper-specific demographic pull
Open with a question or scenario that screams to one persona: "Only 90s kids will remember this puzzle." "If you grew up with [thing], you'll love this." "POV: you have ADHD and..." Filters out non-converters fast = better CTR.

### 10. The reaction face overlay (UGC-style)
Real or AI face reacting in the corner to gameplay. Mimics streamer/UGC content. Works extremely well on TikTok and Reels. **In 2026 this is the dominant format on TikTok for casual.** Use AI avatars from Higgsfield/Heygen if no real talent available.

### 11. The "what would you do?" interactive prompt
Pause on a puzzle mid-solve with 3 visible options A/B/C overlay. Caption "comment your answer." Reveal the right move at 80% timestamp. Then CTA. Works because it converts passive viewers into active ones.

### 12. The trend hijack
Use a trending audio (TikTok), trending visual format (Reels split-screen), or trending meme structure with your gameplay slotted in. Has a 2-week shelf life — burns out fast — but during the burn it's the highest performing format on the platform.

## Genre-specific patterns

### Match-3 (Royal Match, Homescapes, Toon Blast, Match Masters)
- **Hero mechanic:** combo cascades. Show 3+ matches in one move with visible satisfying VFX (sparkles, screen shake, particles).
- **Hero promise:** "rebuild / decorate / save" — the meta-game wrapper, NOT just the matching.
- **Top-performing hooks:** #4 (character in trouble), #6 (transformation), #5 (looks easy → actually hard).
- **Length:** 15-20s sweet spot.

### Hyper-casual (Stickman, Run, Pull-the-pin, Dop)
- **Hero mechanic:** one-finger satisfaction loop, often simulated.
- **Hero promise:** the reaction (audience laugh, character saved, score goes up).
- **Top hooks:** #1 (wrong way), #5 (looks easy), #3 (satisfying ASMR).
- **Length:** 8-15s. Hyper-casual viewers swipe faster.

### Puzzle / brain (Brain Out, Wood Sort, Triple Tile)
- **Hook on the failure state:** show frustration, then relief.
- **Top hooks:** #11 (what would you do), #5 (looks easy), #1 (wrong way).
- **Length:** 12-20s.

### Idle / merge (Merge Mansion, Travel Town, Merge Dragons)
- **Hero mechanic:** progression montage. Bigger, better, more.
- **Top hooks:** #6 (transformation, time-lapse), #4 (character in trouble), narrative cliffhangers.
- **Length:** 20-30s — these games need more setup to read.

### Mid-core / RPG / strategy
- **Hook on power fantasy.** Big armies, big damage numbers, big dragons.
- **Top hooks:** trend hijacks (#12), reaction face (#10), spectacle.
- **Length:** 25-30s.

## Hard rules — what does NOT work in 2026

1. **Pure unrelated fake gameplay.** "Pin-pulling for a match-3" used to print money. Now Apple, Meta, and TikTok flag it as deceptive in policy reviews and store ratings tank. UA teams have moved to *adjacent fake* (still satisfying loop, still in the puzzle universe) or *honest with style*.
2. **Logo-led openings.** Never start with the studio or game logo. The hook must be in frame at frame 1. Logo lives in the CTA frame.
3. **Voice-over in the first 2 seconds.** Sound is off by default on Reels/TikTok auto-scroll. Hook must be visual.
4. **Dense subtitles.** Big simple words only. 3-5 words per beat max. iPhone safe area on top and bottom.
5. **Static text-on-image.** Anything static is instantly skipped. Even a slow camera dolly across a still beats a static frame.
6. **Square-only export.** Always export both 9:16 (TikTok, Reels, Stories) and 1:1 (Feed). Optionally 16:9 for YouTube placements.
7. **Watermarks from generation tools.** Higgsfield, Runway, Kling watermarks make the creative un-shippable. Always export from paid plans or composite over them.

## Naming and tracking convention

Every creative gets a structured filename so we can correlate variants to performance later:

```
<game_code>_<concept>_<hook>_<variant>_<ratio>.mp4

royal-match_hook-wrong-way_v01_9x16.mp4
royal-match_hook-wrong-way_v01_1x1.mp4
royal-match_hook-wrong-way_v02_9x16.mp4    # different camera angle
homescapes_makeover_v01_9x16.mp4
```

Track everything in `directions/creative-tracker.md`:

```markdown
| Date | Game | Concept | Hook | Variants | Posted IG | Posted TT | Notes |
|---|---|---|---|---|---|---|---|
| 2026-04-08 | Royal Match | wrong-way | #1 | 2 | ✅ | ✅ | first ORFEO ref |
```

When a paying client comes in, we add columns: `client`, `delivered`, `IPM`, `CTR`, `D1 retention reported`.

## Reference library to maintain

Build a personal reference library, tag every saved creative with:
- **Game** (Royal Match, Homescapes, ...)
- **Hook pattern** (1-12 from the almanac, can be multi-tagged)
- **Genre** (match-3, hyper-casual, ...)
- **Length**
- **Why I saved it** — one sentence

Sources to pull from regularly:
- **Meta Ads Library**: https://www.facebook.com/ads/library/ — search by Page name (e.g., "Royal Match")
- **TikTok Creative Center**: https://ads.tiktok.com/business/creativecenter/ — Top Ads section, filterable by industry
- **Foreplay.co**: paid, the dedicated UA creative library tool
- **AppMagic** Creatives module: paid, shows top spending creatives by app
- **SensorTower**: paid, similar
- **Pinterest** "casual game ads" boards: free, surprisingly good for art direction reference

Save 5-10 references per week. After 2 months, you have 50-100 organized refs that beat any "creative inspiration" service.

## Working with Higgsfield specifically

Higgsfield.ai (Pavel's primary tool) is best at:
- Cinematic camera moves on AI-generated scenes
- Image-to-video with strong motion control
- Stylized 3D-looking interiors and characters

Higgsfield is **weak** at:
- Maintaining face consistency across cuts (use LoRA in ComfyUI when consistency matters)
- Satisfying matching/cascade VFX (composite particles in CapCut over generated footage)
- Long single takes (it's optimized for 4-10 second clips that you cut together)

Standard workflow:
1. Generate 3-5 scene variants in Higgsfield per beat
2. Pick best one per beat
3. Cut in CapCut to a 15-20s timeline
4. Add SFX, captions, CTA frame
5. Export 9:16 + 1:1
6. Post to IG/TT showcase, also send as DM cold-outreach asset

When a client wants their game's specific style faithfully replicated, switch to ComfyUI with a LoRA trained on their screenshots — see `~/Documents/Claude/Projects/orfeo-creatives/README.md`.
